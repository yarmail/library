Подключаем базу данных

С помощью pgAdmin создаем новую базу данных
Далее в Idea подключаем дополнительную вкладу
View > Tool Windows > Database
(если это необходимо)

А далее во вкладке Database нажимаем на "+", в Меню Data Source
выбираем PostgreSQL.
Заполняем поля во вкладке General:
Database - наша база
User  - логин
Password - пароль
Нажимаем на кнопку Test Connection для проверки
соединения и кнопки Apply и ОК.

Далее во вкладке появляется наша база и во встроенной
консоли мы можем добавлять скрипты для этой базы

---

Доступ к базе данных храним в отдельном файле
name.properties в папке resources.
Данные по подключению к БД можно получить
во вкладке Database по иконке
Data Source Properties

---

Примечания

Предположительно ключ username совпадает
с системной переменной username (в Windows),
отчего возникает ошибка, поэтому желательно использовать
другой ключ, например user_name.

Если возникает необходимость защитить файл name.properties
от передачи в интернет вместе со всем проектом, делают так:
в .gitignore добавляем новое ограничение: *.properties
Далее в папке с оригинальным файлом создаем файл
name.properties.origin, и заполняем его пустыми
ключами без значений, например:
driver=
url=
user_name=
password=
чтобы пользователи знали, какие ключи надо
использовать и делали свои значения

Чтобы файл настроек адекватно относился к русскому языку
я изменил кодировку файлов .properties
Для этого я изменил следующую настройку:
File - Settings - Editor - File Encodings
Default encoding for properties files:
Было: <Properties Default: ISO-8859-1>
Стало: UTF-8
