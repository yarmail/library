Подключаем базу данных

С помощью pgAdmin создаем новую базу данных library
Далее в Idea подключаем дополнительную вкладу
View > Tool Windows > Database
(если это необходимо)

А далее во вкладке Database нажимаем на "+", в Меню Data Source
выбираем PostgreSQL.
Заполняем поля во вкладке General:
Database - наша база
User  - логин
Password - пароль
нажимаем на кнопку Test Connection для проверки
соединения и кнопки Apply и ОК.

Далее во вкладке появляется наша база и во встроенной
консоли мы можем добавлять скрипты для этой базы

---

Доступ к базе данных храним в отдельном файле
database.properties в папке resources.
Данные по подключению к БД можно получить
во вкладке Database по иконке
Data Source Properties


Примечания.
1) Предположительно ключ username совпадает
с системной переменной username (в Windows),
отчего возникает ошибка, поэтому желательно использовать
другой ключ, например user_name.
2) Нельзя передавать файлы настрек и доступы в интернет,
поэтому в .gitignore добавляем новое ограничение
*.properties
Однако, чтобы желающие могли заполнить ключи своими
значениями, передаем пустую форму для заполнения
database.properties.origin,
чтобы они знали, какие ключи используются
